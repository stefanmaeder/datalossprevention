# Grundlagen: TLS Überblick

- ausgeschrieben: Transport Layer Security
- Vorgänger: Secure Sockets Layer (SSL)
- Verschlüsselungsprotokoll zur sicheren Datenübertragung im Internet
- Vertraulichkeit und Integrität zwischen zwei Anwendungen auf Transportprotokoll (z.B. TCP) 
- RFC 5246 (TLS 1.2) primäres Ziel „Datenschutz und Datenintegrität zwischen zwei kommunizierenden Anwendungen “
- RFC 8446 (TLS 1.3) Protokoll, das Client/Server‑Anwendungen eine Kommunikation ermöglicht, „ Entwickelt, um Abhören, Manipulation und Nachrichtenfälschung zu verhindern. “
