# Einleitung

- TLS 1.3: besser für Einzelpersonen – schwieriger für Unternehmen / Unternehmen, die auf Sichtbarkeit für kritische Kontrollen angewiesen sind, bei TLS 1.3 vor erheblichen betrieblichen Herausforderungen stehen
- TLS 1.3 mit Forward Secrecy die gängigen passiven Entschlüsselungstechniken bricht und dass dadurch Funktionen wie Deep Packet Inspection, Security Logging und Diagnostik beeinträchtigt werden.
- Statistik: Anteil der Organisationen weltweit, die einen Verlust sensibler Informationen erlebt haben: Deutschland 57% / 4. Platz weltweit
